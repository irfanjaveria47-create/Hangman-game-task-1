🎮 Hangman Game (Python)

📌 Project Overview

This is a simple command-line Hangman Game developed using Python. The game randomly selects a word, and the player must guess it one letter at a time before running out of attempts.

This project demonstrates the use of Python fundamentals such as variables, loops, conditional statements, functions, lists, strings, and the "random" module.

✨ Features

- Random word selection
- Letter-by-letter guessing
- Tracks correct and incorrect guesses
- Limited number of attempts
- Displays current progress after each guess
- Prevents duplicate guesses
- Simple and beginner-friendly interface

🛠️ Technologies Used

- Python 3.x
- Built-in "random" module

▶️ How to Run

1. Install Python 3 on your computer.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Open the terminal.
5. Run the following command:

python hangman.py

If your system uses:

python3 hangman.py

🎯 How to Play

1. Start the game.
2. Guess one letter at a time.
3. Correct letters are revealed in the hidden word.
4. Incorrect guesses reduce your remaining attempts.
5. Guess the complete word before your attempts run out.
